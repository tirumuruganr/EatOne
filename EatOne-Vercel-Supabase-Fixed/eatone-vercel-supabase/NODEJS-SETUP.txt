EAT ONE Node.js order/invoice workflow

1. npm install
2. Terminal 1: node server.js
3. Terminal 2: npm run dev
4. Website: http://localhost:5173
5. Admin: http://localhost:5173/admin

Customer order IDs: EO-ORD-00001, EO-ORD-00002...
Paid invoice IDs: EO-INV-00001, EO-INV-00002...
The sequences are independent.

This package keeps the original Tailwind/PostCSS configuration and src folder structure.
data.json is for local/testing use. Use PostgreSQL/Supabase for production hosting.
