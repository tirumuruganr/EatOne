IMPORTANT: Run both terminals.

Terminal 1 - backend:
npm run server
Expected: EAT ONE server running on http://localhost:3000

Terminal 2 - frontend:
npm run dev

The Vite dev server now proxies /api requests to port 3000.

Workflow:
1. Customer fills cart and delivery details.
2. Customer clicks Order via WhatsApp.
3. Frontend POSTs the order to /api/orders.
4. Node.js saves the order in data.json and returns EO-ORD-00001, etc.
5. WhatsApp opens with the SAME saved Order ID prominently shown.
6. Copy that Order ID into /admin.
7. Find Order works because the ID exists in data.json.
8. Confirming payment generates a separate EO-INV-xxxxx invoice ID.
